# Delivery App Mini Task

A full-stack Delivery Management System built using React Native, Node.js, Express, and MongoDB for practical assessment.
The system handles order placement, tracking, and delivery status updates using a role-based flow (Customer & Delivery Boy).

---

## 📌 Features

### Customer Side
- Place new orders
- Get unique Order ID after placing order
- Track order status using Order ID

###  Delivery Boy Side
- Secure login
- View all customer orders
- Accept orders
- Update order status:
  - Pending → Accepted → Out for Delivery → Delivered

---

## Order Status Flow
Pending → Accepted → Out for Delivery → Delivered

---

## ⚙️ Tech Stack

### Frontend
- React Native (Expo)
- Axios
- React Navigation

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
